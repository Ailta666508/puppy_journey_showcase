多图垫图 · 风格参考图目录
========================

请将风格参考图放在本目录下，支持 .png / .jpg / .jpeg / .webp / .gif。
图片按【文件名排序】后拼在用户照片后面发给豆包。

顺序与 prompt 对应关系：
- 图2 = 用户上传的真人照片（接口自动放在第 1 位）
- 图3 = 第 1 个文件：肢体结构参考（直立/四肢着地）
- 图4 = 第 2 个文件：02_dog_white.png，小狗比例与画风必须与此图严格一致（白狗）
- 图5 = 第 3 个文件：03_dog_yellow.png，小狗比例与画风必须与此图严格一致（黄狗）
- 图6 = 第 4 个文件：04_example，最终成图效果、比例与风格以此为唯一参考标准

建议命名：01_limb_ref.png  02_dog_white.png  03_dog_yellow.png  04_example.png
